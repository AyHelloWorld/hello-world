用户组和配置文件修改
把当前用户加入到docker用户组中
gswyhq@gswyhq-pc:~$ sudo usermod -a -G docker $USER

添加配置文件
gswyhq@gswyhq-pc:~$ sudo vim /etc/default/docker
在配置文件中添加或修改
DOCKER="/usr/bin/docker"  
重启docker
gswyhq@gswyhq-pc:~$ sudo service docker restart

查看版本：
gswyhq@gswyhq-pc:~$ docker -v
Docker version 1.13.0, build 49bf474

查看状态：
gswyhq@gswyhq-pc:~$ sudo service docker status

其他状态命令：
sudo service docker start    #启动 
sudo service docker stop    #关闭 
sudo service docker restart #重启 

# 有时候pull镜像时，一直都是timeout，这时可采用国内的镜像：
gswyhq@gswyhq-pc:~$ echo "DOCKER_OPTS="$DOCKER_OPTS --registry-mirror=http://f2d6cb40.m.daocloud.io"" | sudo tee -a /etc/default/docker
# 重启docker
gswyhq@gswyhq-pc:~$ sudo service docker restart
# 继续拉取neo4j
gswyhq@gswyhq-pc:~$ sudo docker pull neo4j
